# 文件夹批量创建工具技术开发文档

## 一、项目概述

本项目为一款跨平台（Windows/Mac）桌面应用，用户可通过图形化界面，导入 Excel 模板文件，批量、分层级地在指定目录下创建文件夹，并支持一键撤销本次创建操作。

---

## 二、技术选型

1. **主框架**
   - Electron（跨平台桌面应用开发，兼容 Windows 和 Mac）
2. **前端技术**
   - React（界面开发，组件化，易维护）
   - Ant Design 或 Material UI（UI 组件库，提升界面美观与开发效率）
3. **后端/逻辑层**
   - Node.js（文件系统操作、Excel 解析）
   - xlsx（Excel 文件解析库）
   - fs-extra（增强型文件系统操作库，支持递归创建、删除等）
4. **打包与发布**
   - electron-builder（应用打包与发布）

## 四、主要功能模块设计

### 1. Excel 文件导入与解析模块

- 功能：
  - 允许用户选择并上传 Excel 文件。
  - 使用 xlsx 库解析 Excel 内容，按列识别层级，生成文件夹树结构。
  - 自动去除空行、空列。
  - 仅处理第一个 Sheet。
- 备注：
  - 提供模板下载功能，便于用户规范填写。

### 2. 目标根目录选择模块

- 功能：
  - 允许用户通过文件选择器选择文件夹创建的根目录。

### 3. 文件夹结构预览模块

- 功能：
  - 以树状结构展示即将创建的所有文件夹，供用户确认。

### 4. 文件夹批量创建模块

- 功能：
  - 遍历解析后的树结构，递归创建所有文件夹。
  - 显示创建进度、结果（成功/失败/已存在）。
  - 记录本次创建的所有文件夹路径，便于撤销。

### 5. 撤销操作模块

- 功能：
  - 一键删除本次批量创建的所有文件夹（仅限本次操作，防止误删其他文件夹）。
  - 需处理文件夹非空、部分已被用户手动修改等异常情况。

### 6. 用户界面模块

- 功能：
  - 选择 Excel 文件、下载模板、选择根目录、预览结构、开始创建、显示进度、撤销操作。
  - 交互友好，操作简单。

## 五、关键流程说明

### 1. 文件夹创建流程

1. 用户选择 Excel 文件和目标根目录。
2. 程序解析 Excel，生成文件夹树结构。
3. 用户确认后，点击“开始创建”。
4. 程序递归创建所有文件夹，实时显示进度。
5. 记录所有新建文件夹路径，供撤销使用。

### 2. 撤销流程

1. 用户点击“撤销操作”。
2. 程序按记录路径逆序删除本次创建的所有文件夹。
3. 显示撤销结果。

## 六、异常与边界处理

- Excel 文件格式校验，提示用户按模板填写。
- 文件夹命名非法字符校验（如后续有需求可扩展）。
- 文件夹已存在时跳过或提示。
- 撤销时若文件夹非空或被修改，需提示用户并跳过。
- 大批量操作时需优化性能，避免界面卡顿。
